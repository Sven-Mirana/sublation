# Legal Case Analysis Pattern

## When to use

User provides a case (案号 or materials) and wants structured legal analysis.

## Workflow

1. **Read all case materials** — court judgments, enforcement rulings, retrial petitions, evidence
2. **Build timeline** — chronological sequence of litigation events
3. **Identify legal issues** — map each dispute to applicable statutes (民法典, 公司法, 民事诉讼法, judicial interpretations)
4. **Cross-reference regulations** — check enforcement execution regulations (变更追加规定, 执行异议之诉 rules)
5. **Analyze current posture** — where the case stands, what's the next procedural step
6. **Identify risks** — what arguments could overturn the current favorable outcome

## Key regulations to have ready

- 最高人民法院关于民事执行中变更、追加当事人若干问题的规定 (2020修正)
- 旧公司法司法解释1-5 (for pre-2024 cases)
- 新公司法 (2024施行) — especially 第47条, 第54条, 第88条
- 九民纪要第6条 (出资加速到期)
- 民事诉讼法第211条 (再审事由)

## Pattern for execution-related cases

For enforcement/addition cases (执行追加):
1. Check if company has assets → 终本?
2. Check shareholder capital obligations → 认缴/实缴? 期限届满?
3. Check if acceleration applies → 公司法54条 + 九民纪要第6条
4. Check original shareholder liability → 司法解释三第13条第3款 (发起人资本充实)
5. Check intermediate shareholder → 司法解释三第18条 (knowing transferee)
6. Check new company law applicability → 88条第1款 + 2024.12.24批复 (不溯及)

## Pattern for enforcement objection litigation (执行异议之诉)

1. Verify execution court's addition/change ruling
2. Check if within 15-day filing window (变更追加规定第32条)
3. Identify plaintiff vs defendant roles
4. Assess whether case is formation action (形成之诉) for fee calculation
5. Prepare for retrial if adverse
